\subsection{Detailed Description}
This file contains U\-S\-B core functions not strictly necessary for U\-S\-B to actually operate. This includes\-:


\begin{DoxyItemize}
\item Printing debugging, informational, and error messages
\item Printing the state of the U\-S\-B (through \hyperlink{group__usb_ga264444ef6b00c2562ca7b0d897d34025}{usbinfo()}).
\item Obtaining human-\/readable string descriptors from U\-S\-B devices.
\item Translating certain constants into strings describing them.
\end{DoxyItemize}

To disable all the above functionality, set U\-S\-B\-\_\-\-E\-M\-B\-E\-D\-D\-E\-D to T\-R\-U\-E in usb\-\_\-util.\-h. Obviously, only do this if you know what you're doing.
