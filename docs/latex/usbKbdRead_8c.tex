\hline

Returns \-::\-S\-Y\-S\-E\-R\-R on other error (currently, only if \hyperlink{usbKbdInit_8c_adcec5b3d0038b082ef1865dee38bacf9}{usb\-Kbd\-Init()} has not yet been called).
